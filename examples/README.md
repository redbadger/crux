# Examples

Examples to help you get up to speed with Crux. These are meant to be in order:

1. [Counter](./counter) - get the basics
2. [Counter (HTTP)](./counter-http) - understand how to talk to your API
3. [Weather](./weather) - see how a real app works
4. [Notes](./notes) - another example that demonstrates real time collaboration
5. [Counter (Middleware)](./counter-middleware) - see how to use middleware, an
  advanced way to include rust effects.

See the documentation within each example for more details.
